    Core

    音频加载与元信息读取。

    提供统一的音频读取接口以及基础元数据提取能力，解码依赖 librosa， 无损格式元信息优先使用 soundfile。

    get_audio_info(file_path)

    获取音频文件的基础元数据。

    参数：

    名称 类型 描述 默认
    file_path str

    音频文件路径。

    		 必需

    返回：

    类型 描述
    dict

    元信息字典，包含 sr、channels、duration、samples、 bit_depth 与 format。

    引发：

    类型 描述
    Exception

    获取失败时抛出异常。
    Notes
    • WAV/FLAC/OGG/AIFF 等无损格式使用 soundfile 读取。
    • MP3 等格式通过解码后的样本估算时长与通道数。

    load_audio(file_path, sr=None, mono=True, offset=0.0, duration=None)

    加载音频文件。

    参数：

    名称 类型 描述 默认
    file_path str

    音频文件路径。

    		 必需
    sr int or None

    目标采样率。为 None 时保持原采样率。

    		 None
    mono bool

    是否转换为单声道。

    		 True
    offset float

    起始读取时间（秒）。

    		 0.0
    duration float or None

    读取时长（秒）。为 None 时读取全部。

    		 None

    返回：

    类型 描述
    tuple

    (audio, sample_rate)。mono=True 时 audio 为一维 ndarray， mono=False 时形状为 (channels, samples)，dtype 为 float32。

    引发：

    类型 描述
    Exception

    读取失败时抛出异常。
    Notes
    • MP3 等有损格式的解码依赖系统后端（如 ffmpeg）。
    • 大文件建议使用 offset 与 duration 控制内存占用。

    示例：

    >>> from audiofeatures.core import load_audio
>>> audio, sr = load_audio("example.wav", sr=16000)
>>> audio.shape, sr
((16000,), 16000)

    信号分帧与窗函数处理。

    apply_window(frames, window_type='hann')

    对分帧信号应用窗函数。

    参数：

    名称 类型 描述 默认
    frames ndarray

    二维分帧数组，形状 (n_frames, frame_length)。

    		 必需
    window_type str

    窗函数类型，支持 hann、hamming、blackman、 bartlett、kaiser、rectangular。

    		 'hann'

    返回：

    类型 描述
    ndarray

    加窗后的帧，形状与输入一致。

    引发：

    类型 描述
    ValueError

    输入维度或窗函数类型非法时抛出。
    Notes

    加窗会改变幅值分布，必要时可做能量补偿。

    frame_signal(signal, frame_length, hop_length, center=True)

    将一维信号切分为重叠帧。

    参数：

    名称 类型 描述 默认
    signal ndarray

    一维输入信号。

    		 必需
    frame_length int

    帧长度（样本数）。

    		 必需
    hop_length int

    帧移（样本数）。

    		 必需
    center bool

    是否居中对齐。为 True 时在两端补零。

    		 True

    返回：

    类型 描述
    ndarray

    分帧结果，形状为 (n_frames, frame_length)。

    引发：

    类型 描述
    ValueError

    输入非一维或参数非法时抛出。
    Notes

    当 center=False 且信号长度不足一个帧时返回空数组；center=True 会在两端补零并至少返回一帧。
